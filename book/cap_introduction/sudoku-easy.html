Easy Sudoku Techniques

Learn the basics and start solving!

Welcome, number enthusiast! Remember the golden rule: each row, column, and 3x3 box (the ones with thicker lines) must contain the numbers 1-9, and no number can be repeated within a row, column, or 3x3 box.

The first step is to use pencil marks. In each empty cell, lightly write down all the numbers that could possibly go there. These are your "candidates." To figure out the candidates, look at the numbers already in the cell's row, column, and 3x3 box. Any number already present cannot be a candidate.

1. Singles (Hidden and Naked)

This is where your pencil marks become super helpful! There are two types of Singles:

  • Naked Single: A cell has only one possible number written in it (only one candidate). That number must go in that cell. It's the only option!
  • Hidden Single: A cell might have several candidates, but look closely at its row, column, and 3x3 box. If a number appears as a candidate in only that one cell within that row, column, or box, then that number must go there, even if the cell has other candidates.

Naked Single Example: In the provided grid, let's focus on the third row. We see the numbers 5, 1, 3, 2, 9 and 6 are already filled in. The missing numbers are 4, 7 and 8. For the first empty cell in the third row (the first cell of that row), we see that the number 7 it's the only candidate. Since there's only one candidate, this is a Naked Single, and we can fill in the 7.

Hidden Single Example: Now let's look at the fifth colum(middle one), We see the numbers 6, 3, 9, 1, 5 and 4 are already filled in. The missing numbers are 2, 7 and 8. Only one of the three cells has the number 2 as a candidate, so it's the only cell that the 2 can go, so we can add it. That's a Hidden Single.

Start by looking for Naked Singles across the whole puzzle. Once you've filled those in, check for Hidden Singles in each row, column, and 3x3 box.
